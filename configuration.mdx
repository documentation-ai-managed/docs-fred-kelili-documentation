## Workspace Settings

You define the fundamental characteristics of your documentation space.

<Tabs>
  <Tab title="General" icon="sliders">
    Set workspace name, description, and basic preferences.
    
    | Setting | Description | Default |
    |---------|-------------|---------|
    | Workspace Name | Display name for your documentation space | My Documentation |
    | Description | Brief overview of your workspace | Welcome message |
    | Timezone | Default timezone for timestamps | UTC |
  </Tab>
  <Tab title="Branding" icon="palette">
    Customize colors, logos, and visual elements.
    
    <CodeGroup tabs="CSS,JSON">
      ```css
      :root {
        --brand-color: #3B82F6;
        --heading-color: #1a1a1a;
        --text-color: #374151;
      }
      ```
      ```json
      {
        "brandColor": "#3B82F6",
        "logoUrl": "https://example.com/logo.png",
        "favicon": "https://example.com/favicon.ico"
      }
      ```
    </CodeGroup>
  </Tab>
</Tabs>

## Integrations

You extend functionality through integrations with popular tools and services.

<ExpandableGroup>
  <Expandable title="GitHub Integration" default-open="true">
    Connect your documentation to GitHub repositories for version control and collaboration.
    
    ```javascript
const connectGitHub = async (token) => {
  const response = await fetch('/api/integrations/github', {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${token}` },
    body: JSON.stringify({ repo: 'myorg/docs' })
  });
  return response.json();
};
```
  </Expandable>

  </Expandable>

## Advanced Settings

You access powerful configuration options for complex requirements.

<ParamField query="exportFormat" param-type="string" required="false">
  Specify export format for documentation downloads.
</ParamField>

<ParamField body="customCSS" param-type="string" required="false">
  Custom CSS code for advanced styling.
</ParamField>

```javascript
const updateAdvancedSettings = async (settings) => {
  const response = await fetch('/api/settings/advanced', {
    method: 'PUT',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${localStorage.getItem('token')}`
    },
    body: JSON.stringify(settings)
  });
  return response.json();
};
```

## Backup and Recovery

You protect your documentation with automated backup and recovery options.

| Feature | Description | Configuration |
|---------|-------------|---------------|
| Automatic Backups | Daily backups to cloud storage | Enable in settings |
| Manual Exports | Download full workspace data | Available in admin panel |
| Recovery Points | Restore to specific timestamps | Access recovery panel |
